 AI Market Analyzer & Sentinel
Amazon's AI-powered price monitoring system for opportunity tiering and dashboard creation in Power BI.

##  Technologies
- **Python**: Playwright (Scraping), Pandas (Data), TheFuzz (NLP/Fuzzy Match).

- **Power BI**: Interactive dashboard with AI-powered Top Influencers.

##  How it works
The robot searches for products, corrects typos via AI, classifies prices into tiers (S, A, B, C), and monitors the market in 24-hour loops.
